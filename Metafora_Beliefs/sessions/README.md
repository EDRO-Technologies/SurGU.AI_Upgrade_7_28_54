# Папка сессий

Здесь хранятся записи всех коуч-сессий в формате markdown.

## Формат имени файла
`session_XXXX.md` где XXXX — порядковый номер сессии (0001, 0002, и т.д.)

## Структура файла сессии

```markdown
# Сессия №XXXX

**Дата**: YYYY-MM-DD
**Длительность**: ~XX минут
**Ключевые темы**: [список тем]

## Цель сессии
[Что хотел достичь пользователь]

## Ход сессии
[Основные моменты диалога]

## Выявленные убеждения
- [Убеждение 1] → ID в beliefs.csv: XXX
- [Убеждение 2] → ID в beliefs.csv: XXX

## Инсайты
[Ключевые осознания пользователя]

## Домашнее задание / Практика
[Если было дано]

## Заметки для следующей сессии
[Что важно учесть в будущем]
```

## Конфиденциальность
Все сессии являются конфиденциальными и хранятся локально.
