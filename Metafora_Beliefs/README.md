# Metafora Beliefs — Коуч по трансформации убеждений

Персональный AI-коуч для выявления и трансформации ограничивающих убеждений.

## Структура проекта

```
/
├── CLAUDE.md                    # Системный промпт и инструкции для AI
├── personality_profile.md       # Профиль личности пользователя
├── beliefs.csv                  # База данных убеждений
├── feedback.log                 # Журнал обратной связи
├── README.md                    # Этот файл
│
├── sessions/                    # История всех коуч-сессий
│   ├── README.md
│   ├── session_0001.md
│   ├── session_0002.md
│   └── ...
│
├── knowledge_base/              # Методологии и инструменты коучинга
│   ├── cognitive_distortions.md
│   ├── questioning_techniques.md
│   └── ...
│
└── agents/                      # Специализированные агенты
    ├── coach_prompt.md
    ├── archivarius_prompt.md
    └── ...
```

## Основные компоненты

### 1. CLAUDE.md
Главный системный промпт, определяющий:
- Роль и принципы работы AI-коуча
- Структуру сессий
- Формат коммуникации
- Этические принципы

### 2. personality_profile.md
Динамический профиль пользователя, содержащий:
- Базовую информацию и цели
- Ключевые ценности
- Паттерны мышления
- Прогресс и статистику

### 3. beliefs.csv
База данных всех убеждений:
- Выявленные ограничивающие убеждения
- Статусы трансформации
- Альтернативные поддерживающие убеждения
- Связь с сессиями

### 4. sessions/
Архив всех коуч-сессий:
- Хронология работы
- Ключевые инсайты
- Домашние задания
- Заметки для будущих сессий

### 5. knowledge_base/
Методологическая база:
- Когнитивные искажения
- Техники задавания вопросов
- Коуч-инструменты
- Теоретические материалы

### 6. agents/
Специализированные агенты:
- **Coach**: основной коуч для работы с убеждениями
- **Archivarius**: поиск и анализ информации из истории

## Как начать работу

### Первая сессия
1. Откройте новую сессию с AI
2. Расскажите о себе и своих целях
3. AI обновит personality_profile.md
4. Начнется работа по выявлению убеждений

### Регулярные сессии
1. AI подготовит контекст на основе прошлых сессий
2. Обсуждение актуальных тем и проблем
3. Работа с конкретными убеждениями
4. Сохранение результатов в структуру проекта

### Использование агентов
- `@coach` — для глубокой работы с убеждениями
- `@archivarius` — для поиска информации из истории

## Типы убеждений

Система отслеживает три статуса:
- **Обнаружено** — убеждение выявлено, но еще не прорабатывалось
- **В работе** — активная трансформация убеждения
- **Трансформировано** — сформировано новое поддерживающее убеждение

## Конфиденциальность

Все данные хранятся локально на вашем устройстве. Никакая информация не передается третьим лицам.
